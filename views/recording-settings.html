Recording Settings

Configure recording behavior and route processing parameters

Route Processing

Path Simplification Tolerance
Points within this distance of the simplified line are removed. Lower values preserve more detail but create larger files. Default: 1
meters
Minimum Stop Duration
How long the train must be stationary to count as a stop during automatic recording. Default: 30
seconds
GPS Noise Radius
Maximum distance for coordinates to be considered the same location during stop detection. Default: 10
meters
Minimum Points for Stop
Minimum number of coordinate data points needed to form a valid stop cluster. Default: 10
points

Recording Behavior

Auto-Stop Inactivity Timeout
In automatic mode, recording stops after this period of no movement once past the last timetable time. Default: 120
seconds
Save Frequency
Save recording data to disk every N coordinates. 1 = every coordinate (safest). Higher values improve performance but risk data loss. Default: 1
coordinates

Reset

Reset all recording settings to their default values.
